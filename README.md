# Sign-Up-Page-Simulation
A simple Python project that simulates a User Sign-Up System using Object-Oriented Programming (OOP). The application collects user registration details through the command line, creates a user account object, and displays the registered account information.
## ✨ Features
- User registration
- Collects:
  - First Name
  - Last Name
  - Phone Number
  - Email Address
  - Password
  - Confirm Password
- Displays user account information
- Default account status: **INACTIVE**

## 🛠️ Technologies
- Python 3
- Object-Oriented Programming (OOP)

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/your-username/sign-up-page.git
```

2. Navigate to the project folder:
```bash
cd sign-up-page
```

3. Run the program:
```bash
python main.py
```
